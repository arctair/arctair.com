Storyteller - class project

Storyteller is a web application for teams that storyboard with user stories. Sticky notes are taken to the browser so teams in the same room or different offices can draft stories in real-time. Storyteller offers analysis of user story content to catch common errors like overly-technical language and duplicate or conflicting information.

Storyteller was developed in four months in a team of five student developers as part of the capstone program at NDSU. Technologies used include AngularJS and Firebase.

Minnesota Fall Map - personal project

The Minnesota fall map is a functional example of what can be accomplished using existing APIs and data, GIS concepts and software, cloud computing, and zero operating costs. This map is hosted in Amazon S3 while updated hourly from up-to-date foliage color data from the Minnesota Department of Natural Resources. Updates are performed by a NodeJS function in Amazon Lambda with the help of TurfJS and a theissen polygon generator. Updated geospatial data is pushed to S3 to be retrieved by a static front-end page and drawn with LeafletJS.
